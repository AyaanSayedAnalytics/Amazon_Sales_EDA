📦 Amazon Sales EDA
Exploratory Data Analysis of 3,203 Amazon sales records to uncover profit drivers across categories, products, geographies, customers, and margins. The analysis identified £108k total profit, £2.2k loss leakage across 318 orders, and top SKUs achieving 45%+ margins. All analysis completed using Python, Pandas, Seaborn, Matplotlib.

📊 Key Insights
£108,418.45 total profit generated across the dataset

318 loss‑making orders (≈20% of all orders), totalling £2,200 in losses

Top SKUs achieved 45%+ profit margins, indicating strong product‑level leverage

Category profitability varies sharply, with some categories structurally unprofitable

Geographic performance is uneven, with clear top and bottom regions

Customer profitability is highly skewed, with a small group driving most profit
